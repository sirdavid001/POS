How downloads work

The website and POS app stay separate.

The website portal handles account setup, billing, store details, and downloads. The installed app handles sales, stock, staff work, reports, and daily operations after login.

Create your store account

Register the owner/admin account and store name in the website portal. This does not open the POS app.

Complete activation

Pay the one-time ₦20,000 activation through Paystack or Flutterwave to unlock five months of full access.

Choose your device

After payment is confirmed, the portal shows the real download selector for your device.

Log in inside the app

Install QuickPOS, sign in with the same account, and start using the POS app with active store access.

Release integrity

Installer access is tied to your store status.

Only active admin accounts can request the download manifest. The public website explains the process but does not expose installer links.

Admin-gated access

Managers and cashiers use the installed app. The website portal is reserved for owner/admin account work.

Verified releases

Published builds include version details and checksums where available.

Clear update path

Installed apps can check for supported releases and show update guidance when a new version is ready.
