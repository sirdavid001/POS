Install process

Keep setup clean before the app is opened.

The website handles account setup, payment, staff administration, and installer access. The POS interface only runs inside the installed QuickPOS app.

01

Create or sign in

Use the website account portal with an owner/admin login. Manager and cashier accounts use the installed app.

02

Activate the store

Complete activation in the Account Portal before installer links and device builds become available.

03

Select a device build

Choose the installer that matches the device: Windows, Android, macOS, Linux, or iPhone/iPad.

04

Install and sign in

Open QuickPOS from the installed app and sign in with the same store account to start daily operations.

Release controls

Downloads are protected by account status.

The public website explains the setup process. Actual installer links stay inside the authenticated account portal.

Admin-gated access

Managers and cashiers use the installed app. The website portal is reserved for owner/admin account work.

Verified releases

Published builds include version details and checksums where available.

Clear update path

Installed apps can check for supported releases and show update guidance when a new version is ready.
